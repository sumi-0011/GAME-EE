## 📁 폴더 구조

```
📦src
 ┣ 📂api // API 관련 로직
 ┣ 📂components // 컴포넌트들
 ┃ ┣ 📂SnakeGame // 뱀 게임 관련 컴포넌트
 ┣ 📂constants // 상수 관리
 ┣ 📂hooks // 커스텀 훅
 ┣ 📂pages // 페이지들
 ┃ ┣ 📂game // 게임 페이지
 ┣ 📂services // 서비스 로직
 ┣ 📂styles
 ┃ ┣ 📜colors.ts // 색 관련 상수
 ┃ ┗ 📜theme.ts // 테마
 ┣ 📂types // 타입
 ┗ 📂utils // 유틸 함수
```
